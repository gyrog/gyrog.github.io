fishOrder - Templates screen


Version: 1.4.8

The body of the screen is a list of available template files.

Select a template file by clicking on it, and then clicking the "Ok" button.
Double clicking on a template file, does the same thing.

"Template" panel

The "Edit" button opens the selected template file in a program that can view/edit ".rtf" files. The program used can be defined in "fishOrder Options".
The "Is Default" box is ticked if the selected template file is the default. If it is not ticked, clicking on it will make the selected template file, the new default.

"App" panel

The "Help" button, shows this screen.
The "Cancel" button, closes the screen, ignoring any changes.
The "Ok" button, closes the screen, saving any changes.
