fishOrder - Documents screen


Version: 1.4.8

The body of the screen is a list of the documents currently associated with this occassion.

"List" panel

The "Add" button opens a dialog to select a document, and append it to the list.
The "Remove" button removes the selected document from the list.

"App" panel

The "Help" button, shows this screen.
The "Cancel" button, closes the screen, ignoring any changes.
The "Ok" button, closes the screen, saving any changes.
