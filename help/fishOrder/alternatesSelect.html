fishOrder - Select Alternates screen


Version: 1.4.8

The body of the screen shows the available alternates, each choice in a seperate panel.

Any choice panel

The heading of the panel reflects what is being chosen; e.g. "Title", "First", or "Readings".
In each panel a single item can be selected by clicking on it. The chosen item will be highlighted.
Usually each item is shown as a single line. Sometimes an item is shown as several lines; e.g. "Readings".
When each item is shown using several lines the items are separated by a line containing "--------------------------".
Clicking on any line of the item will highlight the whole item.

"App" panel

The "Help" button, shows this screen.
The "Cancel" button, closes the screen, ignoring any selections made.
The "Ok" button, closes the screen, saving any selections made.

NOTE:

If a "lectionary day" is selected and the "Ok" button clicked, the "Alternates" screen closes automatically.
This is because the other alternates are not valid for the new "lectionary day".
