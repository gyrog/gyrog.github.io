fishOrder - Main screen


Version: 1.4.8

The body of the screen shows a list of field names along with the contents of these fields.

"Manage" panel

The buttons in the 'Manage' panel provide facilites to control a number of things that determine the content of the information displayed.

The "Templates" button, opens a dialog to choose the template for the current document.
The "Calendar" button, opens a dialog to choose a different occasion.
The "Alternates" button, is usually disabled. If enabled, it opens a dialog to choose any of the available alternates.
The "Hymns" button, opens a dialog to select hymns to be used with this document.
The "Documents" button, opens a dialog to manage a list of documents associated with this occasion.

"View" panel

The button in this panel provides the ability to swap between two views:
The "Lectionary" view shows all the common Lectionary based fields.
The "Template" view shows the fields that occur in the current template, in the order they occur in the template.

"Make" panel

The "Document" button causes a document to be generated by merging the field data with the current template.

"App" panel

The "Options" button opens a dialog that allows changing configuration options for the application.
The "Help" button opens this help page.
