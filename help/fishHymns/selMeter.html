fishHymns - Select Meter screen


Version: 1.1.1

Select a meter from the list.

Main body of the dialog

Shows a list of all meters known to fishHymns.
Click on a meter in the list, to select it. Only 1 meter can be selected.
If the required meter is not in this list, then there is unlikely to be a tune for it.

"App" panel

"Help" Shows this screen.
"Cancel" Closes the screen, ignoring any selection.
"Ok" Closes the screen, saving any selection.
