fishHymns - Overview


Version: 1.1.1

"fishHymns" is a program for generating a midi file for a whole hymn (or song).

It uses a collection of tunes which are stored as a midi file for a single verse, and a list of meters it implements.
It uses the meter of the hymn to find a suitable tune, then based on the number of verses it writes a single midi file conatining a suitable number of copies of the tune verse.

Hint: Before any hymn midis can be generated, there needs to be some tunes in the tune collection.

fishHymns - Features

The tempo and midi instrument for the output file can be specified.
A short pause, (of a quarter note duration), can optionally be inserted between each verse.
Facilities to maintain the tune collection are included.

fishHymns - Program interface

Screens in "fishHymns" show the current information in the body of the screen on the left.
On the right, is a column of buttons that provide facilities to manipulate the information on the left.
These buttons are ordered so that the ones towards the top are usually used before the ones towards the bottom.

Screens in "fishHymns" automatically adjust their size to fit the information they contain.
The main screen does not do this, however it's size can be adjusted by dragging it's edges.
Changing the size of the main screen is permanent, (until it is dragged to a different size).
