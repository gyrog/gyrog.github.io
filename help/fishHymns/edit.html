fishHymns - Edit Hymn screen


Version: 1.1.1

A hymn midi file is generated by making multiple copies of a tune midi file.
This dialog provides the facility to enter or change the bits of information used to control this generate process.

When the "Ok" button is clicked, a new midi file is generated for the hymn based on the information provided, overwriting any existing midi file.
So, if the midi file for a hymn is too slow, change the tempo field to a bigger number, then click "Ok".

"Def fields" panel

"Meter" The meter of the hymn is used to find possibly suitable tunes.
Use the "Browse" button to choose an existing meter.
"Verses" The number of verses is used to calculate the default value for the number of times the tune midi is repeated.
"Times" Specifies the number of times the tune midi file will be repeated to generate the hymn midi file.
If "Verses" is defined, this defaults to 1 more than the number of verses.
"Tune" Must be specified. Specifies the name of the midi file that will be copied to produce the hymn midi file.
If the tune has not already been defined in the system, no hymn midi file can be generated.
Use the "Browse" button to select from a list of tunes with the same meter.
"Tempo" Defaults to the tempo of the tune midi file.
If unchanged, the tempo of the hymn midi file will be the tempo of the tune midi file.
If changed, the hymn midi file will use this tempo.
Deleting all characters in the field reinstates the use of the tempo of the tune midi file.
"Instrument" Defaults to the first instrument in the tune midi file.
if unchanged, the instrument(s) in the tune midi file will be used in the hymn midi file.
If changed, all notes in the hymn midi file will be played using this instrument.
Handy if the tune midi plays a piano, and you want the hymn midi file to play an organ.
NOTE: Use the dropdown list to select an instrument or blank.
Selecting the blank option reinstates the use of the tune midi file instrument(s).
"Pause" If ticked, a pause equal in duration to a quarter note is inserted between each copy of the tune midi file.
It's usually easier to sing to, if there is a short pause between verses.
By default, this option is ticked.

"App" panel

"Help" Shows this screen.
"Cancel" Closes the screen, ignoring any changes. No hymn midi file is generated.
"Ok" Closes the screen, saving any changes. A new hymn midi file is generated.
