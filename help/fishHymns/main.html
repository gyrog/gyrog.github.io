fishHymns - Main screen


Version: 1.1.1

"Midi Folder" panel

This shows the full path of the curently selected folder containing the midi files for hymns.
This is where the midi files generated by fishHymns are stored.
The "Browse" button opens a standard folder selector dialog to enable selection of this folder.

The last element of this path is considered to be the collection name.

"Midi Files" panel

This contains a list of all the midi files that have been generated in this folder.
To change an existing file, first click on it in this list, then click the appropriate button in the "Hymns" panel.

"Tunes" panel

The "Tunes" button, opens the tunes management facility.

"Hymns" panel

The "Add" button, begins the process of generating a midi file for a new hymn.
The "Edit" button, opens a dialog to change and regenerate the selected midi file.
The "Play" button, plays the selected midi file.
The "Stop" button, (if enabled), stops playing the midi file.
The "Rename" button, opens a dialog where a new name can be entered for the selected midi file.
The "Delete" button, deletes the selected midi file.

"App" panel

The "Options" button, opems a dialog that allows changing the configuration of the application.
The "Help" button, opens this help page.
