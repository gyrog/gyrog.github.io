fishHymns - Edit Tune screen


Version: 1.1.1

A dialog to enter or change the list of meters that this tune implements.

Unfortunately not all sources of a tune use the same string of numbers to represent a tunes meter.
Having a list of meters allows for the entry of all variants of a tunes meter.

"Meter List" panel

Shows the list of meters currently defined for this tune.

"Meter" panel

"Browse" Opens a dailog to choose an existing meter to be added to the list.
"Add" Opens a standard dialog that allows a new meter to be entered, and added to the list.
"Remove" Removes the selected meter from the list.

"App" panel

"Help" Shows this screen.
"Cancel" Closes the screen, ignoring any changes.
"Ok" Closes the screen, saving any changes.
