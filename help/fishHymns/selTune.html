fishHymns - Select Tune screen


Version: 1.1.1

Select a tune from the list.

Main body of the dialog

Shows a list of all tunes known to fishHymns, with a meter that matches that of the current hymn.
Click on a tune in the list, to select it. Only 1 tune can be selected.
If no suitable tune is found, then either a new tune is required, or this hymn is not played.
Unless there is an acceptable alternate meter that can be specified for this hymn, that does contain a useful tune.

"Midis" panel

"Play" Play the midi file for the selected tune.
"Stop" If active, can be used to stop playing the midi, before it is finished.

"App" panel

"Help" Shows this screen.
"Cancel" Closes the screen, ignoring any selection.
"Ok" Closes the screen, saving any selection.
