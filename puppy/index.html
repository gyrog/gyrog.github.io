Gyrog's Puppy Linux stuff


Minimal Init Overlay

An alternate 'init' for Puppy Linux using overlayfs instead of aufs, and with extra utilities.
Download directory

FrugalPup

A very flexible installer/manager for frugal installs.
Also includes 'StickPup', 'f2StickPup', and 'DiskPup', installers with minimal options.
Download directory

Ziprelease

An alternate way of distributing Puppy, inside a '.zip' file, or inside a '.sfs' file.
Download directory

Firmware

Some large '.sfs' files that contain a rather large set of Linux firmware. Each '.sfs' file has a date in it's filename, so you can easily select the latest.
A script that produces a minimal 'fdrv..sfs' for the current kernel and machine, 'gunzip' the 'mk-local-fdrv.gz' file, and run "./mk-local-fdrv -h".
And a script that produces an 'fdrv..sfs' for the current kernel, 'gunzip' the 'mk-kernel-fdrv.gz' file, and run "./mk-kernel-fdrv -h".
Download directory

Kernels

Example kernels with built-in support for mmc storage devices.
The -gyro- kernels are generated by the updated Woof-CE 'kernel-kit'. They include an 'fdrv...sfs' customised for the particular kernel.
Download directory

Savespec

Savemark on steroids, 'shutdownconfig' writes a 'SAVESPEC' file, rather than a 'SAVEMARK' file, to the install directory.
Also includes a very limited autosave facility.
Download directory

SfsExtra

An alternate to 'SFS-Load on-the-fly' that supports extra sfs files above as well as below the Puppy sfs files.
Also includes 'StackExtra' that supports adding directories to the stack as well as sfs files.
Download directory

InitrdModules

Add vital Kernel modules to the 'initrd.gz' file.
A modified 'init' script, loads these modules before any attempt to access any sfs files.
A kind of replacement for 'initmodules'.
Download directory

PupVault

An encrypted file store using 'LUKS'.
Download directory

Dircrypt

An encrypted file store using 'encfs' or 'gocryptfs'.
Download directory

PCManFM

An alternate filemanager to 'ROX', meant to work alongside 'ROX' rather than replace it.
Download directory

Overlay Init

Remaining files from the 'Overlay Init' project, an exploration of using 'overlayfs' instead of 'aufs'.
Download directory

yad

Both 32bit and 64bit binaries of 'yad'.
Download directory
